## ¿Que Tecnologias usa?

## Scripts para ejecutar

En la carpeta del projecto, deberas ejecutar los siguientes scripts:

    1. npm install
    2. npm run build
    3. npm run dev

### `npm install`

Instala todas las dependencias para el proyecto.

### `npm run dev`

Ejecuta el proyecto.

### `npm run build`

Compila y ejecuta una pagina estatica.
